# RentPe 🏠

RentPe is a full-stack rental marketplace platform that allows users to discover, list, and communicate about rental properties in real time. Inspired by modern rental platforms, RentPe combines secure authentication, property management, user profiles, reviews, and a real-time chat system to provide a complete rental experience.

## 🚀 Live Features

### 🔐 Authentication & Authorization

- Secure user registration and login
- Session-based authentication
- Protected routes and access control
- Owner-only listing management

### 🏡 Property Listings

- Create, edit, and delete property listings
- Upload multiple property images
- Detailed property information
- Property location integration
- Responsive property cards and detail pages

### 👤 User Profiles

- Personalized user profile page
- Profile cover image and avatar
- User listing management
- Review statistics
- Followers and following count
- Clean and mobile-responsive profile design

### ⭐ Reviews & Ratings

- Property review system
- Star-based ratings
- Review management
- User feedback tracking

### 💬 Real-Time Chat System

- WebSocket-powered messaging
- Instant communication between renters and property owners
- Live message delivery without page refresh
- Conversation history support
- Scalable architecture for future enhancements

### 📱 Responsive UI/UX

- Mobile-first design
- Modern and clean interface
- Smooth animations and transitions
- Optimized user experience across devices
